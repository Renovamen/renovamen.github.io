What to install after getting a new Macbook

Jun 28, 2022 · 2 min

  • Code editors
  • VS Code plugins
  • Environments & Package managers
  • Terminal
  • Browsers
  • Note taking
  • Design
  • Others

Apple launched back to school sale recently, so I upgraded from a Macbook Pro 13” 2015 to a Macbook Pro 14” 2021. Though the look is not as attractive as the old one (to me), MBP 14” is still perfect in almost every way (to me).

Anyway, I am trying to write a self-used software checklist when setting up my new laptop.

#Code editors

  • Visual Studio Code
  • Xcode (along with Git)

#VS Code plugins

  • One Dark Pro
  • Dracula
  • Python
  • Volar
  • EditorConfig
  • LaTeX Workshop

#Environments & Package managers

  • Homebrew
  • Miniconda (brew install --cask miniconda)
  • Node.js (along with npm)
  • MacTeX

#Terminal

  • Oh My Zsh
  • zsh-syntax-highlighting
  • zsh-autosuggestions
  • I’ve heard that iTerm2 is great, but I haven’t tried it yet

#Browsers

  • Chrome

#Note taking

  • Typora (Markdown editor)
  • Notion
  • Bear, though it lacks some important features, the bear looks so cute!

#Design

  • Affinity Designer
  • Affinity Photo

#Others

  • Paperlib is a new academic paper management tool

    • Pros

      • It’s simpler than Zotero
      • It’s metadata scraper is tailored for computer science

    • Cons

      • Maintained by a single developer at the moment
      • Bugs occur sometimes (but I’m ok with it)

  • Microsoft Office

  • PDF Reader Pro

  • GitHub Desktop

  • Discord

  • ClashX
