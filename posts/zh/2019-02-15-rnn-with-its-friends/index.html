#梯度消失和爆炸

如果直接把 WhW^hWh 在 ttt 时刻的偏导式展开：

∂Lt∂Wh=∂Lt∂ot∂ot∂ht∂ht∂Wh+∂Lt∂ot∂ot∂ht∂ht∂ht−1∂ht−1∂Wh+...+∂Lt∂ot∂ot∂ht∂ht∂ht−1∂ht−1∂ht−2...∂h2∂h1∂h1∂Wh=\frac{\partial L_t}{\partial W^h} = \frac{\partial L_t}{\partial o_t} \frac{\partial o_t}{\partial h_t} \frac{\partial h_t}{\partial W^h} + \frac{\partial L_t}{\partial o_t} \frac{\partial o_t}{\partial h_t} \frac{\partial h_t}{\partial h_{t-1}} \frac{\partial h_{t-1}}{\partial W^h} + ... + \frac{\partial L_t}{\partial o_t} \frac{\partial o_t}{\partial h_t} \frac{\partial h_t}{\partial h_{t-1}} \frac{\partial h_{t-1}}{\partial h_{t-2}} ... \frac{\partial h_2}{\partial h_1} \frac{\partial h_1}{\partial W^h} =∂Wh∂Lt​​=∂ot​∂Lt​​∂ht​∂ot​​∂Wh∂ht​​+∂ot​∂Lt​​∂ht​∂ot​​∂ht−1​∂ht​​∂Wh∂ht−1​​+...+∂ot​∂Lt​​∂ht​∂ot​​∂ht−1​∂ht​​∂ht−2​∂ht−1​​...∂h1​∂h2​​∂Wh∂h1​​= ∑k=0t∂Lt∂ot∂ot∂ht(∏j=k+1t∂hj∂hj−1)∂hk∂Wh\sum_{k=0}^t \frac{\partial L_t}{\partial o_t} \frac{\partial o_t}{\partial h_t} (\prod_{j=k+1}^t \frac{\partial h_j}{\partial h_{j-1}}) \frac{\partial h_k}{\partial W^h}k=0∑t​∂ot​∂Lt​​∂ht​∂ot​​(j=k+1∏t​∂hj−1​∂hj​​)∂Wh∂hk​​

同理，WxW^xWx 在 ttt 时刻的偏导式展开为：

∂Lt∂Wx=∑k=0t∂Lt∂ot∂ot∂ht(∏j=k+1t∂hj∂hj−1)∂hk∂Wx\frac{\partial L_t}{\partial W^x} = \sum_{k=0}^t \frac{\partial L_t}{\partial o_t} \frac{\partial o_t}{\partial h_t} (\prod_{j=k+1}^t \frac{\partial h_j}{\partial h_{j-1}}) \frac{\partial h_k}{\partial W^x}∂Wx∂Lt​​=k=0∑t​∂ot​∂Lt​​∂ht​∂ot​​(j=k+1∏t​∂hj−1​∂hj​​)∂Wx∂hk​​

hhh 通过激活函数得到，假设激活函数为 tanh：

hj=tanh(WxWj+Whhj−1+bh)h_j = \text{tanh} (W^x W_j + W^h h_{j-1} + b_h)hj​=tanh(WxWj​+Whhj−1​+bh​) ∏j=k+1t∂hj∂hj−1=∏j=k+1t∂hj∂hj−1tanh′Wh\prod_{j=k+1}^t \frac{\partial h_j}{\partial h_{j-1}} = \prod_{j=k+1}^t \frac{\partial h_j}{\partial h_{j-1}} \text{tanh}' W^hj=k+1∏t​∂hj−1​∂hj​​=j=k+1∏t​∂hj−1​∂hj​​tanh′Wh

梯度消失：

可以看到，这俩函数的导数范围都不会超过 1，如果 WhW^hWh 的初始化值也在 [0,1][0, 1][0,1] 之间，那么就是一堆 [0,1][0, 1][0,1] 之间的小数在连乘，乘到最后就会导致梯度越来越接近于 0，造成梯度消失。

在 DNN 中，某一层的梯度消失就意味着那一层的参数再也不更新，那一层的隐层就变成了单纯的映射层。

而 RNN 中，梯度是累加的，就算较远时刻的梯度趋近于 0，累加后的整体梯度依然不会为 0，整体梯度是不会消失的。但这会造成 RNN 被近距离梯度主导，只能利用的有限的历史数据，难以学到远距离的依赖关系。

但相比 sigmoid，tanh 函数的梯度还是更大一点，所以收敛速度要快一些，且引起梯度消失更慢。

而解决梯度消失可以靠换激活函数（ReLU、LeakyReLU、ELU 等）或改传播结构（LSTM、Batch Normalization、ResNet 残差结构）。

因为 yyy 轴右侧导数恒为 1，所以避免了梯度消失的问题。但恒为 1 的导数容易导致梯度爆炸，所以需要一些调参技巧，比如给梯度设定合适的阈值，如果大于这个阈值，就按这个阈值进行更新。

传播流程：

后面公式中的符号说明：

  • σ(⋅)\sigma(\cdot)σ(⋅)：sigmoid 激活函数，会把矩阵转换为一个介于 0 和 1 之间的值作为门控信号，0 表示完全遗忘，1 表示完全接受；
  • tanh(⋅)\text{tanh}(\cdot)tanh(⋅)：tanh 激活函数，会把矩阵转换为一个介于 -1 和 1 之间的值；
  • ⊙\odot⊙：哈达玛积（Hadamard Product），即俩矩阵对应元素相乘，所以要求俩矩阵同形

#遗忘门

Forget Gate，对上一个细胞状态传进来的信息进行选择性遗忘。会根据 ht−1h_{t-1}ht−1​ 和 xtx_txt​ 来为上一个细胞状态 ct−1c_{t-1}ct−1​ 计算一个门控信号，计算公式为：

ft=σ(Wfxxt+Wfhht−1+bf)f_t = \sigma (W_{fx} x_t + W_{fh} h_{t-1} + b_f)ft​=σ(Wfx​xt​+Wfh​ht−1​+bf​)

然后把 ftf_tft​ 跟 ct−1c_{t-1}ct−1​ 相乘，就是最终从上一个状态输入的内容：

ft⊙ct−1f_t \odot c_{t-1}ft​⊙ct−1​

#输入门

Input Gate，对现阶段输入 xtx_txt​ 进行选择性记忆，更新细胞状态。由两个部分构成：

  • sigmoid 激活函数，计算门控信号，控制要记忆哪些内容：

    it=σ(Wixxt+Wihht−1+bi)i_t = \sigma (W_{ix} x_t + W_{ih} h_{t-1} + b_i)it​=σ(Wix​xt​+Wih​ht−1​+bi​)

  • tanh 激活函数，计算现阶段新学到的东西：

    c~t=tanh(Wcxxt+Wchht−1+bc)\tilde{c}_t = \text{tanh}(W_{cx} x_t + W_{ch} h_{t-1} + b_c)c~t​=tanh(Wcx​xt​+Wch​ht−1​+bc​)

这俩相乘后的结果就是最终被记下来的现阶段新学到的东西，再加上从上一个细胞状态输入的内容就是更新后的细胞状态。所以细胞状态的更新公式为：

ct=ft⊙ct−1+it⊙c~tc_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_tct​=ft​⊙ct−1​+it​⊙c~t​

#输出门

Output Gate，现在细胞状态已经更新了，所以要决定那些状态最终会被输出（隐状态 hth_tht​）。依然用 sigmoid 激活函数来计算一个门控信号，控制要输出哪些内容：

ot=σ(Woxxt+Wohht−1+bo)o_t = \sigma (W_{ox} x_t + W_{oh} h_{t-1} + b_o)ot​=σ(Wox​xt​+Woh​ht−1​+bo​)

然后把它跟用 tanh 激活函数放缩过的当前细胞状态 ctc_tct​ 相乘，就是这个阶段最终输出的隐状态：

ht=ot⊙tanh(ct)h_t = o_t \odot \text{tanh}(c_t)ht​=ot​⊙tanh(ct​)

#最终输出

最终的输出 yty_tyt​ 会由 hth_tht​ 变换得到，常见的做法大概是把 hth_tht​ 扔进 softmax：

yt=softmax(ht)y_t = \text{softmax}(h_t)yt​=softmax(ht​)

#GRU

GRU 是 LSTM 的变体。它只有两个门，重置门 rtr_trt​ 和更新门 ztz_tzt​（用一个门达到了遗忘和输入的目的）。它还合并了隐状态和细胞状态。它的模型结构比 LSTM 简单，但同时能达到跟 LSTM 相当的效果。

#重置门

Reset Gate，先计算重置门控信号 rtr_trt​，用于控制要保留上一个时刻的多少信息：

rt=σ(Wrxxt+Wrhht−1+br)r_t = \sigma (W_{rx} x_t + W_{rh} h_{t-1} + b_r)rt​=σ(Wrx​xt​+Wrh​ht−1​+br​)

然后计算当前时刻的候选隐状态（candidate hidden state）：

h^t=tanh(Whxxt+rt⊙(Whhht−1)+bh)\hat{h}_t = \text{tanh} (W_{hx} x_t + r_t \odot (W_{hh} h_{t-1}) + b_h)h^t​=tanh(Whx​xt​+rt​⊙(Whh​ht−1​)+bh​)

相当于 h^t\hat{h}_th^t​ 主要包含了当前输入 xtx_txt​ 的信息，然后有选择性的加入上一时刻的信息（ht−1h_{t-1}ht−1​）。

#更新门

Update Gate，先计算更新门控信号 ztz_tzt​，用于控制要从 ht−1h_{t-1}ht−1​ 中遗忘多少信息和要从 h^t\hat{h}_th^t​ 中记忆多少信息：

zt=σ(Wzxxt+Wzhht−1+bz)z_t = \sigma (W_{zx} x_t + W_{zh} h_{t-1} + b_z)zt​=σ(Wzx​xt​+Wzh​ht−1​+bz​)

然后直接算出当前时刻隐状态 hth_tht​：

ht=(1−zt)⊙ht−1+zt⊙h^th_t = (1 - z_t) \odot h_{t-1} + z_t \odot \hat{h}_tht​=(1−zt​)⊙ht−1​+zt​⊙h^t​

可以理解为 (1−zt)(1 - z_t)(1−zt​) 对标 LSTM 中的遗忘门控，ztz_tzt​ 对标 LSTM 中的输入门控。
